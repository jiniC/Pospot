# Pospot
Pose + posting + spot = traveling app

- 개발 환경:	Android Studio
- Front-end: Java, Android
- Back-end: Django(Python3)
- DB: Realm
- 사용기술<br>
Client: SK T map API, 한국관광공사 Tour API, Retrofit, Gson<br>
Server: Azure Ubuntu VM, Selenium, Chromium
